# dun

Zero friction git commit workflow powered GPT-3.5 ✨
Simply type "dun" instead of  "git add --all && git commit -m ..."

### Install

```
npm i --g dun-cli
echo 'export OPENAI_API_KEY="<Your OPEN_AI Key>"' >> ~/.bashrc
source ~/.bashrc
```

### Use

```
git:(main) dun
```
